# Restaurant DineFlow — Restaurant Setup Checklist

Use this checklist before starting a restaurant trial.

## 1. Basic Details

- [ ] Restaurant name
- [ ] Address
- [ ] Phone
- [ ] GSTIN
- [ ] CGST / SGST
- [ ] Liquor VAT if applicable
- [ ] Service Charge %
- [ ] Google Maps review link

## 2. Super Admin Setup

- [ ] Restaurant added
- [ ] Owner email/password created
- [ ] Plan selected
- [ ] Trial days set
- [ ] Booking link generated
- [ ] Guest menu link generated
- [ ] Payment status set

## 3. Tables

- [ ] Table names added
- [ ] Max pax added
- [ ] Area/section added
- [ ] Status available

## 4. Rooms / Villas

- [ ] Room numbers added
- [ ] Villa names added
- [ ] Prices added
- [ ] Amenities added
- [ ] Images/videos added
- [ ] Max pax/kids added
- [ ] Status available

## 5. Staff

- [ ] Owner login tested
- [ ] Waiter added
- [ ] Captain added
- [ ] Cashier added
- [ ] Chef added
- [ ] Manager/GM added
- [ ] Staff mobile numbers added
- [ ] Staff PIN tested

## 6. Menu

- [ ] Categories added
- [ ] Departments added
- [ ] Menu items added
- [ ] Prices added
- [ ] Veg/non-veg set
- [ ] Liquor items marked
- [ ] Availability checked

## 7. Printer

- [ ] Printer installed on Windows
- [ ] Default printer set
- [ ] Paper size 80mm
- [ ] Printer settings added in DineFlow
- [ ] Department mapped
- [ ] Print Test Receipt done
- [ ] KOT print tested
- [ ] Bill print tested

## 8. Store / Inventory

- [ ] Store departments added
- [ ] Vendors added
- [ ] Raw materials added
- [ ] Opening stock added
- [ ] Minimum stock added
- [ ] Purchase entry tested
- [ ] Stock issue tested
- [ ] Wastage tested
- [ ] Recipe mapping done for important menu items

## 9. Reports

- [ ] Sales Summary tested
- [ ] Date filter tested
- [ ] Payment filter tested
- [ ] Day Close tested
- [ ] Audit Log tested
- [ ] Backup download tested

## 10. APK / Staff App

- [ ] APK installed
- [ ] Staff login tested
- [ ] Notifications allowed
- [ ] Diagnostics checked
- [ ] Test local notification done
- [ ] Ready order notification tested

## 11. Google Automation Optional

- [ ] Google Drive folder created
- [ ] Folder shared with service account
- [ ] Calendar shared with service account
- [ ] Service account JSON saved
- [ ] Room/Villa booking with ID proof tested
- [ ] Calendar event tested

## 12. Demo Final Check

- [ ] POS flow works
- [ ] KOT flow works
- [ ] Bill settle works
- [ ] Print works
- [ ] Store flow works
- [ ] AI Copilot works
- [ ] Guest ordering works
- [ ] Support ticket works
