# Restaurant DineFlow — Demo Script

Use this flow when showing DineFlow to a restaurant owner, accountant, cashier, or manager.

## 1. Start with Landing Page

Say:

> “DineFlow ek AI powered Restaurant POS, Inventory aur Hospitality OS hai. Isme billing ke saath store, reports, rooms, villas, events aur AI Copilot bhi hai.”

Show:
- Landing page modules
- Pricing
- Plan comparison
- Ask DineFlow AI

## 2. Login as Owner

Show:
- Dashboard overview
- Plan status
- Setup wizard
- Quick actions

Say:

> “Owner ko sabse pehle ye checklist dikhegi jisse setup easy ho jata hai.”

## 3. Setup Wizard

Show:
- Restaurant details
- Tables
- Menu
- Staff
- Printer settings
- Store departments
- Demo Data Tools

If demo data needed, click:

> Load Sample Demo Data

## 4. Fast POS

Show:
- Tables / Rooms / Villas
- Running order count
- Running bill amount
- Order button
- Bill button

Say:

> “Rush hour mein cashier/waiter yahin se fast order aur bill manage karega.”

## 5. Order + KOT

Show:
- Take order
- Add items
- Add notes like “less spicy”
- Send KOT
- KOT/Orders tab
- Status: Preparing → Ready → Served

Say:

> “Ready karte hi assigned waiter ko notification/vibration ja sakta hai.”

## 6. Billing

Show:
- Running Bills
- Bill Summary
- Discount
- Service Charge ON/OFF
- Payment mode Cash/UPI/Card/Split
- Print Bill
- Settle Bill
- Settled Bills
- Voided Audit

Say:

> “Accountant ko running, settled aur voided bills alag-alag milenge.”

## 7. Print Station

Show:
- Printer mapping
- Print Test Receipt
- Auto Print ON
- Setup Guide

Say:

> “80mm thermal printer ke liye Print Station aur kiosk mode support ready hai.”

## 8. Store / Inventory

Show:
- Store Dashboard
- Store Departments
- Raw Materials
- Vendors
- Purchase Entry
- Stock Issue
- Wastage
- Recipe Costing
- Stock Audit
- Reports

Say:

> “Yeh module stock leakage aur purchase rate control ke liye hai.”

## 9. Sales Summary

Show:
- Today
- Yesterday
- This Month
- Date range
- Department filter
- Category filter
- Payment filter
- CSV export

Say:

> “Accountant kisi bhi date ki sales, tax aur payment breakup check kar sakta hai.”

## 10. Day Close

Show:
- Opening Cash
- Closing Cash
- Expected Cash
- Cash Difference
- Manager Approval
- Print / Save PDF

Say:

> “Closing ke time cashier ka cash tally yahin se hoga.”

## 11. AI Copilot

Ask:

- Aaj sales summary batao
- Low stock kya hai?
- Vendor payment due batao
- Kya purchase karna chahiye?
- Kaunsa item mehenga ho gaya?

Say:

> “Owner ko daily decision lene ke liye AI assistant milta hai.”

## 12. Guest CRM + Events

Show:
- Guest contacts
- Export CSV
- WhatsApp offer text
- Event/party enquiries

Say:

> “Repeat customers aur party leads yahin se manage honge.”

## 13. APK

Show:
- Staff login
- My Tasks
- Notifications
- Diagnostics

Say:

> “Waiter/Chef/Cashier ke liye app workflow ready hai.”

## 14. Close the Demo

Say:

> “Trial mein full features milenge. Trial ke baad aap apne use ke hisaab se plan choose kar sakte hain.”

Ask:

> “Aapke restaurant mein sabse pehle POS, Store ya Reports mein se kya setup karna chahenge?”
